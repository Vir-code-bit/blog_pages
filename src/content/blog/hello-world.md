# 欢迎来到我的新博客

经过几天的折腾，终于把基于 **Astro** 和 **Material Design 3** 的个人网站搭建起来了。

## 为什么选择 Astro？

Astro 提供了极致的加载速度，默认零 JS，非常适合内容驱动的静态网站。同时它原生支持 Markdown 和 MDX，配合 Shiki 代码高亮，体验极佳。

```javascript
// 这是一个代码块示例
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet('Astro');
```

## 设计理念

本站严格遵循 Google Material Design 3 规范：
- **大圆角**：卡片使用 24px 圆角。
- **Elevation**：通过多级弥散阴影区分层级。
- **色彩**：大面积留白，辅以 Google Blue (`#4285F4`) 作为强调色。
